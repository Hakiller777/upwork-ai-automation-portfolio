# AcmeCRM — Mobile App

## Availability

AcmeCRM is available on iOS (14+) and Android (10+) in the App Store and
Google Play. Search "AcmeCRM" and sign in with your existing workspace
credentials.

## What you can do on mobile

- View and edit contacts and deals
- Log calls, notes, and meetings on the go
- Get push notifications for deal updates and assigned tasks
- View today's tasks and upcoming meetings on the home screen
- Use click-to-call to dial contacts directly from their record

## What's web-only

Billing, user management, automation workflow building, and custom field
configuration are only available on the web app — the mobile app is
optimized for day-to-day selling, not workspace administration.

## Offline behavior

Recently viewed contacts and deals are cached for offline viewing. Changes
made while offline are queued and synced automatically once the connection
is restored; if the same record was also changed elsewhere, the most recent
edit wins.

## Syncing issues

If new changes aren't appearing, pull down on any list screen to force a
manual sync. Ensure the app has background app refresh / background data
permission enabled in your phone's settings, since AcmeCRM relies on this
for push notifications and automatic sync.

## Push notifications

Configure which events send a push notification from the app under
Profile → Notification Settings. This is separate from the web
notification settings, so mobile and desktop alerts can be tuned
independently.
