# AcmeCRM — Reporting & Analytics

## Pre-built reports

- **Pipeline Overview** — total value and deal count per stage, for any
  pipeline and date range.
- **Win Rate** — percentage of closed deals that were won, broken down by
  rep, source, or industry.
- **Sales Velocity** — average time deals spend in each stage, useful for
  spotting bottlenecks.
- **Rep Leaderboard** — deals won, revenue closed, and activities logged
  per rep over a selected period.
- **Forecast** — projected revenue for the current period based on deal
  stage and probability weighting.

## Custom reports

Build a custom report from Reports → New Report by choosing an object
(Contacts or Deals), the fields to display, filters, and a grouping
(by stage, owner, source, or custom field). Custom reports can be saved,
shared with specific team members, and pinned to the dashboard.

## Dashboards

Each user has a personal dashboard; Managers and Admins can also create
team dashboards. Dashboards are a grid of report widgets (charts or tables)
that can be rearranged by dragging.

## Scheduled reports

Any report can be scheduled to be emailed daily, weekly, or monthly to a
list of recipients (they don't need an AcmeCRM account to receive the
email). Configure this from the report's "..." menu → Schedule Email.

## Exporting reports

Every report can be exported to CSV or PDF from the "..." menu on the
report page, in addition to the workspace-wide data export described in
Data Import & Export.

## Data freshness

Reports reflect data as of the last sync, typically within a few seconds of
a change. Scheduled report emails are generated at send time, not cached
from an earlier snapshot.
